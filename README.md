# TypeOfDev Themes

A collection of high-quality dark themes for VS Code.

## Themes included

1. **Admin Type**  
   Inspired by `btop++`. Deep dark grey background with teal/cyan accents. Calm and professional.

2. **Hacker Type**  
   Pure black background with neon green/magenta accents. High contrast, "Matrix" aesthetic.

## Quick install & test (VS Code Insiders)

Option A — Run in Extension Development Host:
1. Open this folder in VS Code.
2. Press **F5**.
3. Select **"Admin Type"** or **"Hacker Type"** from the Color Theme menu.

Option B — Package and Install:
1. `npx --yes vsce package`
2. `code-insiders --install-extension typeofdev-theme-1.0.0.vsix`

## Customize
- **Admin Type**: edit `themes/admin-type.json`
- **Hacker Type**: edit `themes/hacker-type.json`
