## ðŸš€ Installation

Since this theme is currently exclusive to GitHub, follow these steps to install:

1. Go to the **[Releases Page](https://github.com/artyomevgrafov/typeofdev-theme/releases)**.
2. Download the latest `.vsix` file (e.g., `typeofdev-theme-1.0.2.vsix`).
3. Open VS Code.
4. Press `Ctrl+Shift+P` (or `Cmd+Shift+P`).
5. Type **"Install from VSIX"** and select the file you downloaded.
6. Reload VS Code and select your theme!

To tweak the colors yourself:

1. Clone the repo:
   ```bash
   git clone https://github.com/artyomevgrafov/typeofdev-theme.git
   ```
2. Open in VS Code.
3. Press **F5** to launch a preview window with the theme applied.
4. Edit `themes/admin-type.json` or `themes/hacker-type.json` to see changes live.
